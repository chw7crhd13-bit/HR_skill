# HR 薪酬 Excel 使用指南

## 数据处理流程

1. 接收月度薪酬、工资、社保、公积金、奖金或员工花名册文件。
2. 保留原始数据，不直接覆盖源文件。
3. 统一字段名称和格式，例如月份、员工 ID、部门、职级、状态、基本工资、补贴、奖金、社保、公积金、个税。
4. 建立字典表，例如部门表、职级薪酬区间表、员工状态表。
5. 用公式检查数据完整性、重复记录、无效状态和薪酬区间异常。
6. 输出异常清单，交由 HR 或财务进一步确认。
7. 生成部门、月份、职级或城市维度的汇总报表。
8. 对外共享前进行脱敏，只保留必要字段。

## 常用公式

- `VLOOKUP` 或 `XLOOKUP`：用员工 ID、部门或职级匹配字典表。
- `COUNTIFS`：检查员工同月份重复记录，或统计部门异常数量。
- `SUMIFS`：汇总部门薪酬成本、社保成本、公积金成本。
- `AVERAGEIFS`：计算部门、职级、城市维度的平均工资。
- `IF`、`AND`、`OR`：生成行级校验结果，例如 `OK`、`Missing field`、`Out of band`、`Invalid status`。

## 数据校验项目

- 员工 ID 是否为空。
- 同一员工同一月份是否重复。
- 部门是否存在于部门字典表。
- 职级是否存在于职级字典表。
- 员工状态是否属于有效状态。
- 基本工资是否低于或高于职级薪酬区间。
- 实发工资、扣款、个税是否出现明显异常。

## 保密处理

- 不在公开渠道发送原始薪酬明细。
- 共享报表前隐藏姓名、员工 ID 和明细薪酬。
- 只保留分析所需的最小字段集合。
- 原始数据、处理过程文件、最终报表分开保存。
- 临时导出的明细文件在确认后及时删除。
